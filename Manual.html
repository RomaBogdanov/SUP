Общие сведения.

Система построена по трёхзвенной архитектуре: база данных, сервер, клиенты.

Установка.

Общая логика: устанавливаем сервер, разворачиваем базу данных, устанавливаем клиента.
На данный момент клиент и сервер желательно ставить на одну машину.

Установка и запуск сервера.

  • Копируем папку Server по тому пути, по которому хотим разместить наш серверный компонент;
  • Открываем папку Server, запускаем программу SupHost.exe с правами администратора(!!!);
  • Сервер готов к работе.

Установка базы данных.

  • В папке Databases выбираем файл CreateVisitors.sql;
  • Открываем файл CreateVisitors.sql через Microsoft SQL Server Management Studio;
  • Запускаем данный скрипт (нажимаем F5);
  • После выполнения скрипта тестовая база данных готова к работе.

Установка тестового клиента WPF.

  • Копируем папку TestClientWPF по тому пути, по которому хотим разместить клиент;
  • Запускаем программу SupTestClient.exe;
  • Клиент готов к работе.
