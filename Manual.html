Общие сведения.

Система построена по трёхзвенной архитектуре: база данных, сервер, клиенты.

Установка.

Общая логика: устанавливаем сервер, разворачиваем базу данных, устанавливаем клиента.
На данный момент клиент и сервер желательно ставить на одну машину.

Установка базы данных.

  • В папке Databases выбираем файл StartCreatingDataBases.sql;
  • Открываем файл StartCreatingDataBases.sql через Microsoft SQL Server Management Studio и из VS;
  • Запускаем данный скрипт (нажимаем F5);
  • После выполнения скрипта тестовые базы данных готовы к работе.

Установка и запуск сервера.

  • Копируем папку Server по тому пути, по которому хотим разместить наш серверный компонент;
  • Делаем настройку файла SupHost.exe.config - создаём строки подключения к базам данных и задаём логику подключения к серверу(см. рисунки)
  • Открываем папку Server, запускаем программу SupHost.exe;
  • Сервер готов к работе.

Установка тестового клиента WPF.

  • Копируем папку TestClientWPF по тому пути, по которому хотим разместить клиент;
  • Делаем настройку файла SupRealClient.exe.config - надо настроить соединение по WCF;
  • Запускаем программу SupRealClient.exe;
  • Клиент готов к работе.
